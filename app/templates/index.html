The above is a Scheduling app. Given inputs. It creates random preferences and schedules accordingly.

About

  • This app was deployed to Heroku.
  • The scheduler uses a greedy approach:
  • 1. Sort startups by popularity
  • 2. Sort advisor preferences by popularity
  • 3. Schedule the busiest advisors to startups using sorted preferences
  • 4. At given time slot ensure that the column (time) is not occupied
  • 5. If occupied, go to next available time slot.
  • 6. Repeat for all advisors
